# Flow CLI

A CLI tool for flowpad.

## Installation

Install directly from GitHub:

```bash
pip install git+https://github.com/langware-labs/flow-cli.git
```

## Uninstallation

```bash
pip uninstall flow-cli
```

## Usage

```bash
flow
```

This will print "Hello flowpad".

## Development

### Running Tests

Install development dependencies:

```bash
pip install -e ".[dev]"
```

Run tests:

```bash
pytest tests/ -v
```
